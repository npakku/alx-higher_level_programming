#3-say_my_name

==========================
How to use 3_say_my_name
==========================
This module defines a function that says your say_my_name

========
usage
=======

```say_my_name``` prints My name is <first> <last>

::
    >>> say_my_name("Pascal", "Zury")
    My name is Pascal Zury

The parameter ```last name``` is optional. if no string is passed
an empty string is printed
    >>> say_my_name("Pascal")
    My name is Pascal

Invalid names
============
If either first name or last name is not a string TypeError exception is 
raised
